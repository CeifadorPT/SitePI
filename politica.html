Política de Privacidade

Coleta de Dados

Coletamos apenas os dados necessários para o funcionamento do serviço, incluindo:

  • Informações de conta (email, nome)
  • Dados de consumo de energia
  • Configurações de dispositivos

Segurança

Seus dados são protegidos usando:

  • Criptografia de ponta a ponta
  • Autenticação segura
  • Backups regulares

Compartilhamento

Seus dados nunca serão vendidos ou compartilhados com terceiros sem seu consentimento explícito.

Seus Direitos

Você tem direito a:

  • Acessar seus dados
  • Solicitar exclusão
  • Exportar informações
  • Atualizar dados pessoais
